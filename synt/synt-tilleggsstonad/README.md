Spring Boot-utgave av `synt_tilleggsstonad` fra [testnorge-syntetiseringspakker](https://github.com/navikt/testnorge-syntetiseringspakker/tree/master).

Tjenesten eksponerer API-endepunkter for syntetisk generering av tilleggsstonad-vedtak og er satt opp for ONNX-basert modellkjøring.
