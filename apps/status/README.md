# Status Frontend

Applikasjon for viser status på forskjellig applikasjoner og tjeneserter fra Dolly. 

Lenke https://status-dolly.dev.intern.nav.no/ 

## Lokal kjøring
Ha naisdevice kjørende og i "src/main/js" mappe kjør
```
npm install
npm start
```
